# Steganography Tool

Hide and reveal secret messages inside images using LSB (Least Significant Bit) steganography.

## Features
- Hide secret text inside any image
- Reveal hidden messages from images
- Password protection for extra security
- No server needed — runs entirely in browser

## How to Use
1. Go to Hide Message tab
2. Upload an image
3. Type your secret message
4. Download the image with hidden message
5. Share it — only someone with this tool can reveal it!
